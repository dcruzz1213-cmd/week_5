# Week 5 Documentation

## 1. Problem Statement
Create a Python program that calculates a customer's café bill automatically.

## 2. Inputs
- Customer name
- Coffee quantity
- Tea quantity
- Sandwich quantity

## 3. Outputs
- Customer name
- Quantities ordered
- Total bill

## 4. Process Flow
1. Input customer name.
2. Input quantities.
3. Calculate total.
4. Display receipt.

## 5. Constraints
- Quantities must be whole numbers.
- Prices:
  - Coffee = RM8.50
  - Tea = RM6.00
  - Sandwich = RM12.00

## 6. Problem Decomposition
- Get user input.
- Calculate total.
- Print receipt.

## 7. Pseudocode

START

Input customer name

Input coffee quantity

Input tea quantity

Input sandwich quantity

Calculate total

Display receipt

END
